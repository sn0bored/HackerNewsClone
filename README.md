sinatra_skeleton_rakemod
========================

DBC's sinatra skeleton with Rake mods

Usage
-----

    rake db:init

is the same as running
    
    rake db:create
    rake db:migrate
    rake db:seed
    
and

    rake db:reset

is the same as running
    
    rake db:drop
    rake db:create
    rake db:migrate
    rake db:seed
